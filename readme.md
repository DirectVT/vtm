## Features
- User interface
  - True color
  - Kinetic animation
  - Multi-user aware
  - Mouse chording  
- Text processing
  - Unicode clustering
  - Multi-cursor workflow
  - Bidirectional text
  - UTF-8 Everywhere
- Performance and versatility
  - Cross-platform code, C++
  - Non-blocking architecture
  - One third-party dependency
  - Minimum terminal requirements
- Suitability
  - Local usage
  - Remote access
